# 评论规则

小站欢迎大家积极讨论交流！但请注意以下事项：

  • 请勿在评论中写有带政治倾向等非法言论
  • 请仔细思考再发出评论，本站禁止发布垃圾（包括但不限于带侮辱性等）评论
  • 关于添加友链的评论请见 “添加友链”
    如若违反以上规则将会删除并给予警告，多次违反将列入黑名单

# 头像更换

Valine 目前使用的是 Gravatar 作为评论列表头像
请自行登录或注册 Gravatar ，然后修改自己的头像
评论的时候，留下在 Gravatar 注册时所使用的邮箱即可

如果你修改了头像后发现没有更新，请不要慌张，因为 gravatar.cat.net 有七天的缓存期，安静的等待吧～

# 添加友链

欢迎各位小伙伴找我玩！

以下是交换友链的方式以及注意事项

  • 在原则上，本站友链欢迎各类网站（包括但不限于博客 / 同人创作网站主页等），但不接受拥有违法内容的网站
  • 欢迎所有博客类型，无论您是正在学习的新手，还是各种神仙大佬，本站都会收入友链
  • 本站友链采取互换制，除去特殊情况，如果您将我的友链撤下，相应的我也会把您的友链撤下。特殊情况请在评论区回复
  • 如果您的网站无法访问，本站并不会第一时间撤下，而是将您的友链特殊标记，如果您的网站可以再次访问，请在友链评论区回复
  • 为了保持网站规范，以及方便友链的添加，请您不要在除友链外其他页面评论区发布友链请求
  • 友链添加成功后，如果您在评论时注明了邮箱，您可以在评论区获得” 老婆！”（waifu）的标签
  • 本条例可能不定期进行修改，但不会影响已经添加的友链
